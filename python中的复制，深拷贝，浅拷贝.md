#  python中的复制

对象赋值实际上是对象的引用。

```
a=[1,2,3,[3,4,5]]

b=a

```

直接赋值,默认浅拷贝传递对象的引用而已,原始列表改变，被赋值的b也会做相同的改变

```
a.append(5)

print (a)

a=[1,2,3,[3,4,5]]

print(b)

a=[1,2,3,[3,4,5]]

```

#  python深拷贝

深拷贝，包含对象里面的自对象的拷贝，所以原始对象的改变不会造成深拷贝里任何子元素的改变

```
d=[1,2,3,4,[5,6]]
e=copy.deepcopy(d)

d.append(5)
d.[4].append(7)

print(e)
[1,2,3,4,[5,6]]
```
